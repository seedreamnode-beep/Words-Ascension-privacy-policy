🔒 隱私權政策 Privacy Policy

最後更新日期：2026年4月25日

本應用程式（以下簡稱「本App」）非常重視使用者的隱私。以下說明我們如何處理您的資料：

1. 資料收集

本App 不會收集任何足以辨識個人身分的資料，如姓名、地址、電話或聯絡人清單。您輸入的文字僅在即時 AI 處理過程中使用，不會被開發者讀取、儲存或用於任何訓練。

2. 本地儲存

本App 使用裝置本地儲存（SharedPreferences）保存以下資訊，這些資料僅存在於您的裝置上：

  • 魔法值餘額與使用紀錄
  • 訂閱方案狀態
  • 魔法文庫（轉換歷史，僅存在本機）

3. Firebase 雲端服務

本App 使用 Google Firebase 提供以下功能：

  • Firebase Auth 處理 Google / Apple 帳號登入，及匿名登入
  • Cloud Firestore 同步您的魔法值與訂閱狀態（與帳號綁定）
  • Cloud Functions 安全代理 AI 文字生成請求，保護 API 金鑰

Firebase 相關資料受 Google 隱私政策約束。

4. AI 文字生成（Google Gemini API）

您輸入的原始文字將透過 Cloud Functions 安全傳送至 Google Gemini API 進行轉換處理。開發者無法讀取您的輸入內容，生成完成後資料不會被保留。詳情請參閱 Google AI 服務條款。

5. 廣告與追蹤

本App 不包含任何廣告，亦不使用任何第三方追蹤工具追蹤您的個人行為。

6. 資料刪除

刪除本App 即可移除所有本地資料。如需刪除 Firebase 帳號相關資料，請聯繫 seedream.node@gmail.com，我們將協助處理。

7. 政策更新

本隱私權政策可能不定期更新，修訂後將更新本頁面日期並於 App 內公告。
